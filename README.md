# SRMS-OOP-Project
Student Record Management System- Java OOP project.


## Project Overview
The Student Record Management System (SRMS) is a Java-based console application developed as part of the IT214 course. The system manages university student records by allowing users to add, update, delete, search, and display student information. It also supports file storage and report generation.

## Features
- Add, update, delete, search, and display student records
- Save and load student data using text files
- Generate reports by GPA and department
- Exception handling for invalid input
- Multithreading for auto-saving student data

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Exception Handling
- Multithreading
- File Handling
- GitHub

## How to Run
1. Open the project in a Java IDE (NetBeans, IntelliJ, or VS Code).
2. Compile and run the `Main` class.
3. Follow the on-screen menu instructions.
